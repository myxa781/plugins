Murglar Plugins

неофициальный каталог плагинов для Murglar — мощного Android плеера, созданного для аудиофилов и любителей музыки.

🎵 Что такое Murglar?

Murglar — это Android-плеер нового поколения, разработанный с акцентом на качество звука и гибкость настройки. Архитектура плагинов позволяет подключать любые музыкальные сервисы без изменения основного приложения.

Плагины написаны на JVM-совместимых языках (Kotlin/Java) и подключаются в виде APK или JAR файлов. SDK версии 9 обеспечивает стабильный API для разработчиков.

⚡ Возможности каталога

🔍
Живой поиск
Мгновенный поиск по названию, описанию и тегам плагинов без перезагрузки страницы.
🎛️
Фильтры и сортировка
Фильтрация по категории, типу и SDK. Сортировка по дате, названию и версии.
📱
PWA
Каталог работает как Progressive Web App — устанавливается на экран и работает офлайн.
📦
Прямая загрузка
APK и JAR доступны напрямую без регистрации. SHA256 хэши для верификации.
🖼️
Галерея скриншотов
Просмотр скриншотов каждого плагина с поддержкой lightbox, клавиатуры и свайпа.
📋
История изменений
Полная история версий каждого плагина с описанием изменений.

➕ Как добавить новый плагин

Добавление нового плагина в каталог не требует правки HTML. Достаточно выполнить четыре шага:

Шаг 1 — Папка
Создать папку assets/plugins/plugin-name/ со скриншотами
Шаг 2 — Файлы
Положить plugin.apk, plugin.jar и icon.png
Шаг 3 — JSON
Добавить запись в data/plugins.json по образцу
Шаг 4 — Деплой
Запушить в GitHub — плагин появится в каталоге автоматически
